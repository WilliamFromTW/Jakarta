How To Compile

1. j2se version 7 or above

2. jakarta poi 3.9 final

3. apache commons net 1.2.2

4. gradle 2.6 or above
   gradle java => generate jar file
   gradle javadoc => generate java document files

6. if use test sample file , create tmp directory in the same drive root directory ( for example:  d:/tmp ) 
   and copy test files  in directory "test_files" to "tmp" directory.
